**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 2:** [Learning and practicing EC2, VPC, AWS networking services; configuring VPN; project team meeting.](1.2-week2/)

**Week 3:** [Learning AWS Compute VM Services and Practicing AWS Backup, File Storage Gateway, and Autoscaling Group](1.3-week3/)

**Week 4:** [Learning AWS Storage Services and Practicing VM Import/Export, File Storage Gateway](1.4-week4/)

**Week 5:** [Learning AWS Security Services and Practicing Security Hub, EC2 Cost Optimization, and Resource Management](1.5-week5/)

**Week 6:** [Practicing IAM & KMS Labs and Learning AWS Database Services](1.6-week6/)

**Week 7:** [Practicing Module 6 Labs and Working on the Group Project](1.7-week7/)

**Week 8:** [Reviewing and Preparing for the Midterm Exam](1.8-week8/)

**Week 9:** [Continuing Review and Preparation for the Midterm Exam](1.9-week9/)

**Week 10:** [Researching and Developing the Group Project in First Cloud Journey](1.10-week10/)

**Week 11:** [Continued progress in researching and implementing the group project.](1.11-week11/)

**Week 12:** [Continued progress in researching and implementing the group project.](1.12-week12/)
